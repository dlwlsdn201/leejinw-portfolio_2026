# 프로젝트 설명

사용자가 가진 재료 정보를 기반으로 구글 Gemini API를 연동하여 맞춤형 레시피를 제안해주는 서비스입니다.

## 핵심 경험

- Edge Function 환경(Vercel Serverless)에서 AI 프롬프트 튜닝 경험
- Markdown 파싱을 통한 응답 UI 렌더링
- 사용자 피드백(좋아요/별점) 반영 시스템 구축
