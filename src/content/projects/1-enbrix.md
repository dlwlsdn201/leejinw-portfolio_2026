# 프로젝트 설명
기존 레거시 React SPA를 Next.js App Router 기반으로 마이그레이션하여 **초기 로딩 속도를 40% 이상 개선**하고, SSR을 적용하였습니다.

## 주요 성과
- Zustand를 활용한 글로벌 상태 관리 단순화
- TailwindCSS와 Shadcn UI 기반의 사내 디자인 시스템 컴포넌트 라이브러리 개발 및 적용
- 차트 렌더링 최적화를 통한 메모리 누수 방지
