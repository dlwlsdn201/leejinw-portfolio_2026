# 프로젝트 설명

Todomorrow는 매일의 할 일에 집중하고, 지나간 기록과 다가올 내일을 아주 게으르고 즉흥 성향이 강한 사람도 최소한의 노력으로 관리할 수 있도록 돕는 미니멀리즘 투두(To-Do) 애플리케이션입니다.

## 핵심 경험

- Supabase 를 통해 BaaS 기반의 백엔드, DB 서비스 통합 구축 및 시스템 구성 경험
- SNS 프로파일 기반의 데이터 스키마 설계 경험
- 첫 svelte 기반의 소규모 프로젝트 개발 경험
- Cursor AI agent 명세 문서 기반의 바이브 코딩 경험
